# C# Password Generator

## Overview

Генератор паролей на C# - это консольный инструмент, который позволяет пользователям с легкостью создавать надежные и защищенные пароли. 
Пользователи могут настраивать такие критерии пароля, как длина, включение прописных /строчных букв, цифр и специальных символов.

## Features

- Генерируйте случайные пароли с заданными критериями.
- Выберите длину пароля.
- Включайте или исключайте заглавные буквы, строчные буквы, цифры и специальные символы.
- Просмотрите сгенерированный пароль в консоли.
- Скопируйте сгенерированный пароль в буфер обмена.

## Prerequisites

- [Visual Studio](https://visualstudio.microsoft.com/) должен быть установлен.

### Installation

1. Клонировать репозиторий:

    ```bash
    git clone https://github.com/sagyndykovaaida/passwordGeneration.git
    
    ```

2. Откройте проект в Visual Studio.

3. Создайте и запустите проект с помощью Visual Studio.

## Usage

1. Запустите приложение с помощью Visual Studio с помощью команды ctrl+F5 или кнопки "Пуск(run)" 
2. Следуйте инструкциям на экране, чтобы настроить свой пароль.
3. Просмотрите сгенерированный пароль в консоли.
4. При необходимости скопируйте пароль в буфер обмена.

## Configuration

- Измените класс `PasswordGenerator` в `Password Generator.cs`, чтобы изменить настройки по умолчанию.
